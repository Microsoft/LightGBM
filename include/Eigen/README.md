**Eigen is a C++ template library for linear algebra: matrices, vectors, numerical solvers, and related algorithms.**

For more information go to http://eigen.tuxfamily.org/.

The files in this folder are a subset of the full Eigen library: this folder contains only the files needed by LightGBM. The Eigen files used here are all licensed under MPL2. See COPYING.MPL2 for the full license.
